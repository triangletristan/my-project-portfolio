Driving Chaffeur Project Link

This project was a project for the Congressional App Challenge. Me and my partner would spend a bunch of time on this, planning each of the sprites and what the app would do. This app would teach people the basics of driving, allowing them to go through lessons and do a test drive.

"Grandma's House" Project (2022)

"Grandma's House" Project Link

A text adventure project involving a visit to your grandmothers house. You are put here as a result of your attitudes towards your parents during the summer. Discover secrets that your grandmother has in her house to find out the truth behind everything. This project is unfinished as I only got the first part of it done.

Sharpen Project (2023)

Sharpen Project Link

This project was a project for the AP Create Task made on Snap. This app would test peoples brain and motor skills in a multitude of games.Some of these games would range from: Aim Pop, a point and click game, Type Racer, a typing test game, and Cave Memory, a memorization game. With all these games, the app was practically three apps in one, with a bunch of different pixel art sprites. I honestly could've made an app with just one game but it's too late.
